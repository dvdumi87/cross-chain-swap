# Hardhat Project
## Deploying custom USDC and WETH ERC20 contracts

Supported networks:
- Sepolia
- Etherlink Testnet
- X Layer Testnet
- Local Erigon Devnet


```shell
# deploy USDC
npx hardhat run scripts/deploy-usdc.js --network <net>
# deploy WETH
npx hardhat run scripts/deploy-weth.js --network <net>
```
